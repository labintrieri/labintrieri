## Investigações com dados

### [Hackers em sites gov.br](https://www1.folha.uol.com.br/mercado/2024/06/hackers-poem-tigrinho-links-suspeitos-e-termos-sexuais-em-sites-de-governos.shtml)
Levantamento revelou quase 9.000 links com domínio gov.br invadidos por hackers, apresentando termos relacionados a apostas, pornografia e até abusos de menores.
**Metodologia:** Raspagem automatizada + filtros com 41 termos → [Ver repositório](https://github.com/labintrieri/sites-apostas-govbr)

### [Fortune Tiger no YouTube](https://www1.folha.uol.com.br/mercado/2024/08/youtube-abriga-3500-videos-de-propaganda-enganosa-de-tigrinho-e-caca-niqueis-online.shtml)
Encontrei mais de 3.500 vídeos com propaganda enganosa de jogos de cassino online, violando regras de jogo responsável.
**Metodologia:** Extração automatizada + análise amostral com IA → [Ver repositório](https://github.com/labintrieri/fortune-tiger-youtube)

### [Transparência na Defensoria de SP](https://www1.folha.uol.com.br/poder/2025/12/defensoria-de-sp-admite-erro-em-dados-de-pagamentos-e-culpa-base-preenchida-a-mao.shtml)

### [Envelhecimento do PT](https://www1.folha.uol.com.br/poder/2025/07/envelhecimento-e-concentracao-regional-marcam-pt-que-escolhe-novo-comando.shtml)
